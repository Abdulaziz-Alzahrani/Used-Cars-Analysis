this is a project done to perform analysis on used cars dataset from [Kaggle] (https://www.kaggle.com/ammaraahmad/used-cars-dataset)
## Installation

you will need the following libraries:

### 1- panadas
```bash
pip install pandas
```
### 2- numpy
```bash
pip install numpy
```
### 3- matplotlib
```bash
pip install matplotlib 
```
### 4- seaborn
```bash
pip install seaborn
```
